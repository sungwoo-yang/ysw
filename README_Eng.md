##  Controls

- Left / Right Movement: A (left), D (right)
- Jump: W or Space
- Interaction: F (read signboards, activate campfires, etc.)
- Dash: Tap Shift briefly to dash in that direction
- Run: Hold Shift for more than 0.5 seconds to start running
