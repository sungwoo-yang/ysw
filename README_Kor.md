## 게임 소개
OLLIM(올림)은 알 수 없는 이유로 떨어진 12개의 별자리를 구하는 별의 정령이라는 컨셉을 가지고 있습니다, 거대한 맵을 돌아나디며 퍼즐을 풀고 보스방에 도달하여 빛을 반사하는 퍼즐 형식의 플레이로 별자리를 구원해야합니다.

## 게임 플레이 가이드
- 현재 맵은 튜토리얼입니다.
- 표지판을 읽으면서 조작법을 익혀보세요.
- 보스전 방식의 퍼즐을 구현해두었습니다.
- 퍼즐을 클리어 해보세요.


##  조작법

- 좌우 이동: A (왼쪽), D (오른쪽)
- 점프: W 또는 Space
- 상호작용: F (표지판 읽기, 모닥불 활성화 등)
- 대쉬: Shift 키를 짧게 누르면 해당 방향으로 빠르게 대쉬합니다.
- 달리기: Shift 키를 0.5초 이상 누르고 있으면 달리기를 시작합니다.



# 별 (Star) 과 빛 (Light)

별: 게임 세계에는 '빛을 발사하는 별'과 '죽은(빛이 필요한) 별'이 있습니다.

빛 발사: 발사하는 별은 몇 초 간격으로 두 종류의 빛(레이저)을 쏩니다.

🟡 노란빛: 뱀처럼 느린 속도로 다가옵니다. 이 빛은 패링(반사)할 수 없습니다. 피해야 합니다.

🔴 빨간빛: 즉시 생성되며 플레이어를 향해 날아옵니다. 이 빛은 방패로 '패링'해야 합니다.

# 빛 퍼즐 방식

방패 생성: 마우스 오른쪽 버튼 (우클릭)을 누르면 마우스 커서 위치에 방패가 생성됩니다.

느린 이동: 방패를 들고 있는 동안(우클릭을 누르고 있는 동안) 플레이어의 이동 속도가 느려집니다.

패링 (빛 반사):

우클릭으로 방패를 생성한 후 0.5초 이내에 '빨간빛'이 방패에 닿으면, 빛을 성공적으로 반사(패링)할 수 있습니다.

이 반사된 빛을 이용해 '죽은(빛이 필요한) 별'을 활성화하거나 다른 장치들을 조작하며 게임을 행해야 합니다.
